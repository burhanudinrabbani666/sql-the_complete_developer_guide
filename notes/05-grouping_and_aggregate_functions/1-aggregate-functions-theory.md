# Aggregate Functions Theory

## Terminology

Aggregate: Mathematical operation returning a single (aggregated) result
Function: Predefined operations, which can be executed on demand

Example:

```sql
SELECT SUM(salary) FROM employees
```

## Core Aggregate Function Overview

```sql
COUNT()
SUM()
MAX()
MIN()
AVG()
```
