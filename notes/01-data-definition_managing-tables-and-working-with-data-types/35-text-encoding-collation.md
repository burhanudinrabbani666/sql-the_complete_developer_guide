# text encoding collation

## Text Values

- Encoding:
  - Which chracter are supported and can be stored?
  - Not all Encoding support all characters -> Encoding can be set on database, table or column level.
- Collation:
  - How are characters compared?
  - Different collations control "who wins" in comparisons -> Collation can be set on databse, table or column level.
