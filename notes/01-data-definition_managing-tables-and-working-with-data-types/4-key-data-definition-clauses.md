# Key data definition clauses

## Managing Database & Tables

Data Defintion Operations ==> Typically executed way less frequentlu than data manipulation operations / queries

- Creating Database

```sql
CREATE DATABASE
```

Configure name and base settigs, e,g. regarding text handling

- Creating Tables

```sql
CREATE TABLE
```

Configure name and fields + field value types + extra settings (e,g. constraints, keys)

- Updating Database

```sql
ALTER DATABASE

```

Update Configuration

- Creating Tables

```sql
ALTER TABLE
```

Update Configuration
