# Data normalization first steps

### Why are we spliting data?

- Data should be normalized
- a concept that reduces data redudancy and increas data maintability
- Goal: Split compound and grouped data into multiple, standalone values

Example:

- Full Name: "Burhanudin Rabbani" ❎

- firstName: "Burhanudin" ✅
- lastName: "Rabbani" ✅
