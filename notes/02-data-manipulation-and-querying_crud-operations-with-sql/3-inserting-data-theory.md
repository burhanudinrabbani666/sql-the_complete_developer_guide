# Inserting data theory

Inserting values into columns (column names can be omitted)

```sql
INSERT INTO <table name> (<column name>)
VALUES (<column values>);
```

---

Inserting multiaple rows in one single command

```sql
INSERT INTO <table name> (<column name>)
VALUES
    (<column calues>), -- row 1
    (<column calues>); -- row 2
```
